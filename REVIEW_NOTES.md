# Stav této verze

Tato verze rozšiřuje původní interchange tracker o veřejně dohledávané scheme fees, processing fees a merchant service charge.

Hlavní změny:

- automatizace změněna na měsíční;
- odstraněna závislost na Anthropic API;
- přidán Brave Search API a bezklíčový GDELT;
- přidán vícejazyčný slovník pro EU/EEA;
- přidán trvalý archiv mediálních a veřejných pozorování;
- přidány měsíční trendové grafy;
- přidán CEE filtr;
- přidáno dynamické skrývání prázdných síťových a Media sloupců;
- přidán transparentní výpočet „Odhadovaný reálný poplatek“;
- dashboard používá Arial a rozšířené barevné rozlišení vrstev.

Živý síťový sběr nelze ověřit v prostředí bez běžného přístupu k internetu. Po nahrání na GitHub je proto nutné ručně spustit první Action a zkontrolovat log i výsledné zdroje.
